# M.O.B.I.™ · Git Convention v1.4.0

### 1. The Sovereign Protocol
The M.O.B.I.™ Grand Fleet prioritizes security and ownership of the delivery pipeline. As such, the **SSH protocol** (`git@github.com:...`) is the mandatory standard for all repository interactions.
- **Preference:** Always use SSH over HTTPS.
- **Rationale:** SSH provides a more secure, non-interactive, and agent-friendly tunnel for automated synchronization and deployment.

### 2. Commit Standards
- **Format:** Conventional Commits are mandatory.
- **Types:** `feat`, `fix`, `docs`, `style`, `refactor`, `perf`, `test`, `chore`.
- **Scope:** Always include the version or task ID in parentheses, e.g., `feat(v1.4.0): describe change`.
- **Atomic Commits:** One logical change per commit.

### 3. Branching Strategy
- **Master Branch:** `main`.
- **Feature Branches:** `feat/vX.X.X-description`.
- **Fix Branches:** `fix/vX.X.X-description`.
- **Docs Branches:** `docs/vX.X.X-description`.
