# Specification v1.3.7: Mobi Design Kit Expansion & Analytics

## Overview
This specification covers the introduction of essential business-ready design system components (Modal, Toast, Checkbox, Table, Tabs), secondary layout helpers (Badge, Skeleton, Tooltip, Pagination), and deep Google Analytics 4 (GA4) system hooks.

---

## Changelog

### Added (Core Design Kit)
- **MobiModal** — [NEW] Highly accessible portal overlay featuring deep focus-trapping (`tabindex` management, boundary elements), backdrop dismissal, Escape key bindings, and size modifiers (`sm`, `md`, `lg`, `xl`, `full`).
- **MobiToastProvider** / **useMobiToast** — [NEW] Context-driven micro-notifications system providing success, error, warning, info, and customizable banner toasts. Supports individual dismiss timers, and automatically unbinds timers on component unmount.
- **MobiCheckbox** — [NEW] Triple-state input representing checkbox, radio buttons, and animated toggles. Fully styled with standard CSS color tokens.
- **MobiTable** — [NEW] Strongly-typed tabular display with custom cell renders, sorting/alignment configuration, custom empty states, and built-in skeleton rows.
- **MobiTabs** — [NEW] Responsive navigation menu with three design variants (`default`, `pills`, `underline`), disabled tabs support, badge counters, and keyboard arrows navigation.

### Added (Layout Helpers & Accessibility)
- **MobiBadge** — [NEW] Compact color capsules representing semantic alerts (`default`, `success`, `error`, `warning`, `info`, `outline`) with optional status indicators.
- **MobiSkeleton** / **MobiSkeletonGroup** — [NEW] Pulse placeholder layouts indicating async card structures.
- **MobiTooltip** — [NEW] Contextual helper bubbles featuring 4 positioning directions (`top`, `bottom`, `left`, `right`), hover delay triggers, and screen-reader anchors.
- **MobiPagination** — [NEW] Intuitive page browser calculating complex sibling arrays with edge buttons and navigation ellipsis layout blocks.

### Added (System Telemetry)
- **MobiAnalytics** — [NEW] Headless dynamic injection wrapper loading the Google Analytics 4 tag with CSP nonce and user opt-out features.
- **useMobiAnalytics** — [NEW] Fully-typed analytics helper hook containing custom event trackers (`trackEvent`) and customer profiling parameters.

## API Specification

### `MobiModal`
```tsx
<MobiModal
  isOpen={isOpen}
  onClose={() => setIsOpen(false)}
  title="Critical Reactor Check"
  size="md"
>
  <p className="text-sm">Please verify subsector matrix values.</p>
</MobiModal>
```

### `useMobiToast`
```typescript
const { toast } = useMobiToast();
toast.success('Grid voltage stabilized.', 'Success');
```
