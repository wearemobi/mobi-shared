## Context & Goal
Integrate the official hero branding assets into the shared library and refine the user identity header for better spatial efficiency and professional UX. This update ensures that all Grand Fleet applications can render the primary logo with theme-awareness and entrance animations.

## Changelog

### Added
- **`MobiLogoHero`** — new theme-aware logo component. Automatically resolves `light` / `dark` variants and handles `system` preference changes. Includes a 700ms entrance animation.
- **`MobiLogoHero.md`** — API documentation for the new component.

### Changed
- **`MobiUserBadge`** — added `name` prop. In `expanded` mode, the user's full name is now prioritized as the primary title, with the email moving to the subtitle position.
- **`MobiUserBadge`** — standardized avatar initials to 1 letter across all variants for visual consistency.
- **`MobiSentinelMenu`** — updated `MobiSentinelMenuUser` interface to include the `name` field.
- **Documentation** — updated `README.md`, `MobiSentinelMenu.md`, and `MobiUserBadge.md` to reflect new parameters.
