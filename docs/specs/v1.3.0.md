## Context & Goal
Enable the **Agentic Orchestration Protocol** by transitioning the `MobiChatWidget` from a "Self-Managed" standalone component to a "Controlled" orchestratable asset. This allows external systems (Dashboards, Sidebars, Navbars) to dictate the assistant's state, ensuring a unified user experience across the M.O.B.I. Grand Fleet.

## Changelog

### Added
- **Controlled State Management** — Introduced `isOpen` prop to `MobiChatWidget` for external control.
- **Orchestration Callbacks** — Added `onToggle` handler to synchronize internal UI triggers (Close button, Sentinel click) with parent application states.
- **Sentinel Persistence** — Enhanced state preservation during external toggle transitions.

### Changed
- **Internal Refactoring** — Optimized the widget's internal state logic to support both "Autonomous" and "Controlled" modes seamlessly.
- **API Surface Alignment** — Standardized prop naming conventions for better DX across the fleet.
