## Context & Goal
Enable the **Agentic Orchestration Protocol** by allowing external systems to control the visibility and state of the `MobiChatWidget`. This shift from "Self-Managed" to "Controlled" mode allows for tighter integration between dashboard menus and the support assistant.

## Changelog

### Added
- **Controlled Mode** — `MobiChatWidget` now supports the `isOpen` prop for external state management.
- **State Synchronization** — Added `onToggle` callback to sync internal widget actions (Close button, Trigger click) with external parent states.

### Changed
- **`MobiChatWidget` Internal Logic** — Refactored to handle both internal (default) and controlled (prop-driven) states seamlessly.

## Files Modified

## Acceptance Criteria

- [x] Internal Close button correctly triggers the `onToggle` callback.
