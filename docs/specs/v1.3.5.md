# <img src="https://wearemobi.com/icon-light.svg" width="24" height="24" valign="middle"> M.O.B.I.™ · Language Switcher Component v1.3.5

## Context & Goal
Provide the M.O.B.I.™ platform with a premium, modular language selector to toggle between English and Spanish. This replicates the clean design aesthetics, structure, and internal state hook support model of `MobiThemeSwitcher`, showing bold EN / ES texts instead of flags or long text options.

## Changelog

### Changed
- **MobiSentinelMenu** — Integrated the newly created `<MobiLangSwitcher />` under the configurations area of the dropdown. It is fully controlled by the `lang` and `onLangChange` parent props, defaulting to internal uncontrolled state if omitted. The layout was upgraded into a premium settings panel utilizing left-aligned uppercase monospace labels (TEMA, IDIOMA) and right-aligned switchers to establish perfect visual symmetry.

### Added
- **MobiLangSwitcher** — [NEW] Standalone, premium selector supporting controlled and uncontrolled state options for English (`en`) and Spanish (`es`) selections. It renders matching HSL/CSS variables and standard rounded borders, aligning in perfect visual symmetry with the theme switcher.

## API Specification

### `MobiLangSwitcher`
Can be instantiated standalone anywhere in the host application to provide independent language controls:

```tsx
import { MobiLangSwitcher } from '@wearemobi/shared';

// Uncontrolled (Defaults to EN)
<MobiLangSwitcher />

// Controlled (Binds to custom host states)
<MobiLangSwitcher 
  lang="es" 
  onChange={(nextLang) => console.log(nextLang)} 
  className="my-custom-layout-class"
/>
```
