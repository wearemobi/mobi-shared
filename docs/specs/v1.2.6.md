## Context & Goal
Consolidate UI components from specialized modules into the core shared library. This release focus on "Asset Containers" and "Ingestion Vectors" by migrating `MobiCard` and `MobiDropbox` from the FE chassis into the `@wearemobi/shared` DNA.

## Changelog

### Added
- **`MobiCard`** — A premium asset container with a dedicated header, status indicators (pulse), and technical metadata footer.
- **`MobiDropbox`** — High-performance file ingestion component with drag-and-drop support, extension filtering, and visual feedback states.

### Changed
- **Version Bump** — Incremented library version to `v1.2.6` to reflect new component availability.
