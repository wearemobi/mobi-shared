# Specification v1.3.8: Loader, Step-by-Step Wizard & ListView

## Overview
This specification details the introduction of visual status loaders, guided step setup procedures, and unified card/list representations of records.

---

## Changelog

### Added
- **MobiLoader** — [NEW] Visual loader component featuring the official Mobi logo and multiple animation layouts:
  * `spinner` (radial loading ring rotating around the mark).
  * `pulse` (interactive scale-ping glow indicators).
  * `bars` (pulsating audio/telemetry vertical wave animation).
  * `screen` (absolute fullscreen overlay blurred glass backdrop).
- **MobiWizard** / **useMobiWizard** — [NEW] Process onboarding container. Keeps absolute design consistency with MobiCard and MobiButton elements, including:
  * A structured header showing overall and current step completion.
  * Validation rules hook preventing transition triggers if standard prerequisites fail.
  * Controlled Back, Next, Cancel, or Submit buttons footer.
- **MobiListView** — [NEW] Stacked item list display. Supports:
  * List borders or individual distinct rounded Card variants.
  * Structured icon slots (`leading`), actions tags (`trailing`), headings (`headline`), and descriptive summaries (`description`).
  * Click events, selection, and disabled status layouts.

## API Specification

### `MobiLoader`
```tsx
import { MobiLoader } from '@wearemobi/shared';

// Regular inline spinner
<MobiLoader variant="spinner" size="md" label="CALIBRATING REACTORS..." />

// Fullscreen screen overlay
<MobiLoader variant="screen" label="SYNCHRONIZING SECURE TUNNELS..." />
```

### `MobiWizard` & `useMobiWizard`
```typescript
import { useMobiWizard, MobiWizard } from '@wearemobi/shared';

const steps = [
  { id: '1', title: 'Initialize Reactor', content: <StepOne />, isValid: true },
  { id: '2', title: 'Assign Captains', content: <StepTwo />, isValid: false }
];

const wizard = useMobiWizard({
  totalSteps: steps.length,
  onComplete: () => console.log('Matrix Loaded!')
});

<MobiWizard
  steps={steps}
  activeStepIndex={wizard.activeStep}
  onNext={wizard.nextStep}
  onPrev={wizard.prevStep}
  isFirstStep={wizard.isFirstStep}
  isLastStep={wizard.isLastStep}
/>
```

### `MobiListView`
```tsx
import { MobiListView } from '@wearemobi/shared';

const items = [
  {
    id: '1',
    headline: 'Core Reactor Vanguard',
    description: 'Subsector 4-A operating at 92.4% yield.',
    leading: <ReactorIcon />,
    trailing: <Badge variant="success">Nominal</Badge>,
    onClick: () => selectReactor('1')
  }
];

<MobiListView items={items} variant="cards" />
```
