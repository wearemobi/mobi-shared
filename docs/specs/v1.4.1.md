# Specification v1.4.1 - Sentinel Navigation Component

## Overview
This specification defines the `MobiSentinelNav` component, a tactical navigation orchestration layer designed for the M.O.B.I.™ Grand Fleet. It provides a seamless transition between desktop sidebar layouts and mobile hamburger menus, ensuring high accessibility and module-to-module sovereignty.

## Architectural Mandate

### 1. Hybrid Responsive Design
- **Desktop (Laptops/Monitors)**: Displays as a persistent, high-density sidebar with micro-animations.
- **Mobile (Smartphones/Tablets)**: Collapses into a tactical hamburger trigger that opens a full-screen or slide-out module drawer.
- **Breakpoints**: Standardized at `768px` (md) for layout transition.

### 2. Component Structure (`MobiSentinelNav`)
The component will be composed of three internal functional layers:
- **Trigger Layer**: The hamburger/close icon toggle for mobile states.
- **Manifest Layer**: The list of navigation items (modules), supporting icons and status badges.
- **Identity Layer**: Displays the active Island/Sector DNA at the bottom of the navigator.

## API Definition

### `MobiSentinelNavProps`

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `items` | `NavModule[]` | Yes | Array of navigation module objects. |
| `activeId` | `string` | Yes | ID of the currently active module for highlighting. |
| `onNavigate` | `(id: string) => void` | Yes | Callback triggered upon module selection. |
| `isCollapsed` | `boolean` | No | Initial state for desktop minimized sidebar. |
| `showDNA` | `boolean` | No | Whether to show the Island/Sector metadata (Default: `true`). |
| `className` | `string` | No | Additional styling overrides. |

### `NavModule` Interface
```typescript
interface NavModule {
  id: string;
  label: string;
  icon: React.ReactNode;
  path?: string;
  badge?: string | number;
  disabled?: boolean;
  isExternal?: boolean;
}
```

## UI/UX Specifications

### Interactions
- **Hover States**: Sublte glow effects using `mobi-primary` accents.
- **Transitions**: Smooth `300ms` cubic-bezier transforms for sidebar expansion and drawer slides.
- **Glassmorphism**: Backdrop blur effects (`12px`) for the mobile drawer to maintain visual depth.

### Visual Tokens
- **Background**: `var(--mobi-surface-dark)` with `0.8` opacity for glass effects.
- **Border**: `1px solid var(--mobi-border)`.
- **Text**: `var(--mobi-text-muted)` shifting to `var(--mobi-text)` on active/hover.
